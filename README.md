# 🚗 Driver Drowsiness Detection System

A real-time driver monitoring system that detects drowsiness using computer vision.

## 📌 Features
- Real-time face detection
- Eye detection using Haar Cascades
- Alert when eyes are closed
- Sound alarm for warning

## 🛠️ Technologies Used
- Python
- OpenCV

## 🚀 How It Works
The system captures live video from a webcam, detects the driver's face and eyes, and triggers an alert when eyes are closed, indicating drowsiness.

## ▶️ How to Run
1. Install dependencies:
   pip install opencv-python playsound

2. Run the program:
   python main.py

## ⚠️ Note
This is a basic prototype and can be improved using deep learning models for higher accuracy.
